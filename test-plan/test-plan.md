# Test Plan – SauceDemo Manual Testing Project

## Objective
The objective of this testing project is to validate the core functionality of the SauceDemo web application through manual testing practices.

---

## Application Under Test
Application: SauceDemo  
URL: https://www.saucedemo.com/

---

## Main User Flow
Valid user login → Browse inventory → Add products to cart → Access cart → Complete checkout

---

## Modules Identified
- Authentication
- Inventory
- Cart
- Checkout
- Side menu
- Product filter

---

## Risk Prioritization
Highest priority modules:
1. Login
2. Checkout
3. Cart
4. Inventory

Reason:
These modules directly affect the main purchasing flow and core user access.
