TEST SCENARIOS

LOGIN

- Verify that a valid user can log in with valid credentials
- Verify that a user cannot log in with invalid credentials
- Verify that a locked out user cannot log in
- Verify that a user cannot log in when the username field is empty
- Verify that a user cannot log in when the password field is empty
- Verify that a user cannot log in when both fields are empty
- Verify that the password field masks typed characters

INVENTORY

Filter
- Verify that products can be sorted correctly by price
- Verify that products can be sorted correctly in alphabetical order

Add / Remove
- Verify that products can be added to the cart from the inventory page
- Verify that products can be removed from the cart from the inventory page

Product Information
- Verify that each product card displays image, title, description, and price
- Verify that product information is displayed consistently and without spelling issues
- Verify that clicking on a product opens the product details page

Visual/UI
- Verify that all product cards are properly aligned
- Verify that product cards do not overflow or break the layout

CART

- Verify that products can be removed from the cart page
- Verify that users can proceed to checkout from the cart page with products in the cart
- Verify that users cannot proceed to checkout with an empty cart
- Verify that cart items are displayed correctly
- Verify that cart information persists after navigation
